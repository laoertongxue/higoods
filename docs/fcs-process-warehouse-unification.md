# FCS 待加工仓 / 待交出仓统一模型

本文件记录前端原型中的统一仓模型口径。当前项目没有后端、服务端、数据库；印花、染色、特殊工艺、后道工厂的待加工仓、待交出仓、交出记录和审核记录统一由前端本地事实源承接。

## 流程图一：统一待加工仓流程

加工单
-> 生成待加工仓记录
-> 工厂接收
-> 工厂执行
-> 更新待加工仓状态
-> 加工完成
-> 生成待交出仓记录

## 流程图二：统一待交出仓流程

待交出仓记录
-> 工厂发起交出
-> 生成交出记录
-> 接收方回写
-> 数量一致
-> 进入下一环节或完成

## 流程图三：统一差异流程

交出记录
-> 接收方回写
-> 数量不一致
-> 生成差异记录
-> 平台处理差异
-> 重新交出或确认差异
-> 继续流转

## 流程图四：印花仓记录流程

印花加工单
-> 印花待加工仓
-> 打印和转印
-> 印花待交出仓
-> 印花交出记录
-> 印花审核记录
-> 印花完成

## 流程图五：染色仓记录流程

染色加工单
-> 染色待加工仓
-> 打样备料和染缸执行
-> 染色待交出仓
-> 染色交出记录
-> 染色审核记录
-> 染色完成

## 流程图六：特殊工艺仓记录流程

特殊工艺单
-> 特殊工艺待加工仓
-> 绑定菲票
-> 完成特殊工艺
-> 特殊工艺待交出仓
-> 特殊工艺交出记录
-> 数量变化和差异记录
-> 特殊工艺完成

## 流程图七：后道仓记录流程

后道单
-> 后道待加工仓
-> 接收领料
-> 质检
-> 后道
-> 复检
-> 后道交出仓
-> 后道交出记录
-> 接收方回写
-> 后道完成

## 流程图八：车缝厂已做后道仓记录流程

车缝后道任务
-> 车缝工厂执行后道
-> 交给后道工厂
-> 生成后道工厂待接收领料记录
-> 后道工厂质检
-> 后道工厂复检
-> 后道交出仓
-> 后道交出记录
-> 后道完成

## 状态机一：统一待加工仓记录

待入仓
-> 已入仓
-> 已接收
-> 加工中
-> 加工完成
-> 已转待交出仓
-> 已关闭

异常流转：

已接收
-> 有差异
-> 平台处理中
-> 已接收

## 状态机二：统一待交出仓记录

待交出
-> 已部分交出
-> 已全部交出
-> 待回写
-> 已回写
-> 已关闭

差异流转：

待回写
-> 有差异
-> 平台处理中
-> 已回写或重新交出

## 状态机三：统一交出记录

待回写
-> 已回写
-> 已关闭

差异流转：

待回写
-> 有差异
-> 平台处理中
-> 已关闭或重新交出

## 状态机四：后道仓记录

待后道
-> 后道中
-> 后道完成
-> 待质检
-> 质检中
-> 质检完成
-> 待复检
-> 复检中
-> 复检完成
-> 待交出
-> 已交出
-> 已回写
-> 已完成

## 状态机五：非专门工厂后道仓记录

待后道
-> 后道中
-> 后道完成
-> 待交后道工厂
-> 已交后道工厂
-> 后道工厂待质检
-> 质检中
-> 质检完成
-> 待复检
-> 复检中
-> 复检完成
-> 后道交出仓
-> 已完成
