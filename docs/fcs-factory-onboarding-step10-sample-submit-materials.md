# 工厂入驻口径收口第 3 步：样衣提交资料扩展

## 背景

本次只扩展工厂端样衣提交资料和平台样衣审核兜底资料，不调整入驻申请表单、不调整审核结果口径、不调整 PPIC 逻辑，也不调整正式转档逻辑。

## 工厂端提交资料

工厂在“待工厂提交样衣审核”或“样衣审核退回”状态提交资料时，资料分为三组：

- 样衣资料：样衣照片、样衣视频、工艺说明、问题说明、备注、补充文件。
- 工厂资料：工厂照片、工厂视频。
- 老板身份资料：老板身份证号码/护照号码、老板身份证复印件或照片。

其中样衣照片、样衣视频、工艺说明、工厂照片、工厂视频为必填。老板身份证号码/护照号码、老板身份证复印件或照片在工厂端为可选。

## 平台审核兜底

平台样衣审核仍只有“已通过 / 未通过”两个结果。

- 选择“已通过”时，老板身份证号码/护照号码和老板身份证复印件或照片必须齐全。
- 如果工厂已提交身份资料，平台只读展示。
- 如果工厂未提交或只提交一部分，平台审核人员在审核弹窗补录缺失部分。
- 选择“未通过”时，不强制补齐老板身份资料，仍按需重新提交内容退回。

## 身份资料来源

系统记录身份资料来源：

- 工厂提交
- 平台补录
- 工厂提交和平台补录

同时记录补齐时间和补齐人，样衣审核记录会留存审核时的老板身份资料快照。

## 流程图

工厂提交样衣审核  
-> 上传样衣照片和样衣视频  
-> 上传工厂照片和工厂视频  
-> 可选填写老板身份资料  
-> 提交到平台审核  
-> 平台审核  
-> 若已通过则校验老板身份资料是否齐全  
-> 不齐全则平台补录  
-> 齐全后审核通过

## 状态图

待工厂提交样衣审核  
-> 待平台审核样衣

待平台审核样衣  
-> 老板身份资料不齐全  
-> 平台补录  
-> 样衣审核通过待转正式

待平台审核样衣  
-> 未通过  
-> 样衣审核退回
