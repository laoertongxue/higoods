# 纸样捆条与重复纸样校验

## 业务范围

本轮只处理技术包纸样管理中的两个收口点：

1. 捆条跟随纸样维护，不再作为裁片明细逐片特殊工艺维护。
2. 同一技术包下保存纸样前先做重复校验，避免重复上传同一个纸样。

本轮不调整工序工艺字典、BOM 缩水 / 洗水联动、纸样两步维护结构、DXF/RUL 解析算法、裁片明细片数列、逐片特殊工艺、菲票生成和特殊工艺任务生成。

## 捆条维护

捆条属于工序工艺字典中的辅助工艺，适用对象为面料。它跟随纸样维护，一个纸样可以维护多个捆条。

每个捆条包含：

- 捆条编号
- 捆条名称
- 长度（cm）
- 宽度（cm）
- 备注

长度和宽度必须大于 0，单位固定为厘米。捆条保存到纸样记录的 `bindingStrips`，并进入技术包版本快照。

## 逐片特殊工艺边界

捆条不属于裁片逐片特殊工艺：

- `canSelectInBindingArea = true`
- `canSelectInPatternPiece = false`
- 裁片明细逐片特殊工艺选择器不展示“捆条”

## 重复校验范围

保存纸样前执行 `checkDuplicatePattern`。

明确重复会阻止保存：

- 同名纸样
- 相同 PRJ 文件
- 相同 DXF 文件
- 相同 RUL 文件

疑似重复会提示用户确认：

- 相同关联物料和纸样类型
- 相同唛架图片
- 相似解析结构

疑似重复确认后，纸样保存 `duplicateConfirmed` 和 `duplicateWarningReasons`，用于后续追溯。

## 捆条维护流程

```text
进入版师技术信息
-> 添加捆条
-> 填写捆条名称长度宽度
-> 保存纸样
-> 保存捆条到纸样
-> 技术包版本快照保留捆条
```

## 重复校验流程

```text
点击保存纸样
-> 校验必填字段
-> 校验捆条明细
-> 校验同名纸样和重复文件
-> 明确重复则阻止保存
-> 疑似重复则提示确认
-> 用户确认后保存为新纸样
```

## 状态机

```text
待维护
-> 捆条已添加
-> 捆条已校验
-> 纸样重复校验通过
-> 已保存
```

## 重复异常状态机

```text
纸样重复校验
-> 明确重复
-> 阻止保存
-> 返回修改
```

```text
纸样重复校验
-> 疑似重复
-> 待用户确认
-> 返回修改或继续保存
```

## 版本快照

技术包版本快照保留：

- `bindingStrips`
- `patternSignature`
- `duplicateConfirmed`
- `duplicateWarningReasons`
