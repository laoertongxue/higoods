## 2. 设计与交互自查

- 继续复用现有矩阵详情渲染、抽屉、目标确认和历史事件处理，不复制矩阵模板。
- 列表保留当前筛选、排序和滚动状态；点击“查看矩阵”通过原生 `<a target="_blank" rel="noopener noreferrer">` 打开新窗口，避免异步 SPA 事件处理阻断浏览器弹窗。
- URL 同时携带 `productionOrderId` 与 `productionOrderNo`，新窗口首次加载和刷新均按参数恢复同一生产单矩阵。
- 新窗口只展示矩阵详情区，仍保留“查看更新历史”、部位详情、目标选择和补料跳转等既有业务动作。
- 详情页顶部新增“返回裁片放行管理”、一级标题“裁片放行矩阵详情”和生产单 / SPU / 款式摘要；返回入口只在当前详情窗口内回到列表，不影响原列表窗口。
- 生产单、矩阵版本、当前齐套、冻结/持续更新来源状态均在详情首屏可见；颜色尺码物料细节通过既有点击/焦点抽屉查看。
- 文案、状态、单位均为中文；沿用企业后台表格、卡片、抽屉和状态颜色规范。
