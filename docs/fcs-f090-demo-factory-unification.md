# FCS 演示工厂 F090 统一说明

本轮只统一演示工厂口径和列表展示，不调整状态机、移动端写回、仓记录业务流、交出记录业务流、统计口径或打印服务。

统一演示工厂：

- 工厂编号：F090
- 工厂名称：全能力测试工厂
- 页面展示：全能力测试工厂（F090）

## 覆盖范围

- 印花加工单、印花执行节点、印花仓记录、印花统计与移动端演示任务。
- 染色加工单、染色执行节点、染色仓记录、染色统计与移动端演示任务。
- 裁片原始裁片单、合并裁剪批次、裁片仓记录、裁片 PDA 任务与裁片现场页面。
- 特殊工艺任务单、特殊工艺待加工仓、待交出仓、交出记录、菲票流转与移动端演示任务。

## 中文流程图

生产单
-> 生成印花染色裁片特殊工艺加工单
-> 统一演示工厂为全能力测试工厂F090
-> 绑定移动端演示任务
-> 工艺工厂运营系统展示工厂名称和编号
-> 工厂端移动应用展示同一工厂
-> 工厂生产协同系统展示同一工厂

## 中文状态机

待统一
-> 已统一工厂主数据
-> 已统一加工单工厂
-> 已统一移动端任务工厂
-> 已统一列表展示
-> 已完成校验

异常流转：

已统一加工单工厂
-> 移动端任务工厂不一致
-> 修正移动端任务工厂
-> 已完成校验

## 展示规则

加工单、任务单、仓记录、移动端任务卡片与详情中的工厂字段优先展示为：

全能力测试工厂（F090）

如页面已有独立工厂编号列，也可以拆成：

- 工厂：全能力测试工厂
- 工厂编号：F090

本轮不改变真实业务规则，不表示真实生产中同一工厂一定能承接所有工艺；这里只是为了原型联调演示统一数据口径。
