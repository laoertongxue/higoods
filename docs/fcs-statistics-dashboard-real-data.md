# FCS 统计与大屏真实执行数据

本文档记录印花、染色、特殊工艺、后道统计与大屏统一取数口径。当前项目仍是前端原型，不引入后端、数据库或外部接口；统计结果均由前端统一事实源派生。

## 状态机

### 状态机一：统计数据更新状态

初始数据
-> 执行动作写回
-> 仓记录更新
-> 交出记录更新
-> 差异记录更新
-> 审核记录更新
-> 统计重新计算
-> 页面同步展示

### 状态机二：印花统计覆盖状态

待打印
-> 打印中
-> 打印完成
-> 待转印
-> 转印中
-> 转印完成
-> 待送货
-> 待交出
-> 待审核
-> 已完成

异常流转：

待审核
-> 已驳回
-> 待交出

交出差异流转：

待回写
-> 有差异
-> 平台处理中
-> 已回写或需重新交出

### 状态机三：染色统计覆盖状态

待样衣
-> 打样中
-> 打样完成
-> 待原料
-> 备料完成
-> 待排缸
-> 已排缸
-> 染色中
-> 脱水中
-> 烘干中
-> 定型中
-> 打卷中
-> 包装中
-> 待送货
-> 待交出
-> 待审核
-> 已完成

交出差异流转：

待回写
-> 有差异
-> 平台处理中
-> 已回写或需重新交出

### 状态机四：特殊工艺统计覆盖状态

待加工
-> 加工中
-> 加工完成
-> 待交出
-> 已交出
-> 已完成

差异流转：

已交出
-> 有差异
-> 平台处理中
-> 已确认差异或需重新交出

### 状态机五：后道统计覆盖状态

待后道
-> 后道中
-> 后道完成
-> 待质检
-> 质检中
-> 质检完成
-> 待复检
-> 复检中
-> 复检完成
-> 待交出
-> 已交出
-> 已回写
-> 已完成

非专门工厂流转：

后道完成
-> 待交后道工厂
-> 后道工厂待质检

## 取数约束

- 统计核心函数集中在 `src/data/fcs/process-statistics-domain.ts`。
- 统计派生层只读取统一加工单、统一仓记录、统一交出记录、统一差异记录、统一审核记录，以及特殊工艺菲票和后道执行记录。
- 印花大屏使用印花统计派生数据，不再使用固定样例数据。
- 染色统计统一使用“染色统计”文案，不再使用旧名称。
- 统计只按“接收领料、质检、后道、复检、交出”覆盖后道任务，不把开扣眼、装扣子、熨烫、包装作为后道任务动作。
- 统计与大屏不生成质量扣款流水、对账流水或结算流水。
