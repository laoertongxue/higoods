# FCS 工厂端移动应用执行回写

本文件记录工厂端移动应用执行动作如何写回 Web 端加工单、执行进度、待交出仓、交出记录和审核视图。当前项目没有后端、服务端和数据库，所有演示数据通过前端 mock 与本地内存模型串联。

## 流程图一：移动端执行写回通用流程

加工单
-> 进入待加工仓
-> 工厂端移动应用接收任务
-> 工厂端移动应用执行节点
-> 写回 Web 端加工单详情
-> 执行完成
-> 生成待交出仓记录
-> 工厂端移动应用发起交出
-> 接收方回写数量
-> 数量一致进入下一环节
-> 数量不一致生成差异记录
-> 平台处理差异
-> 继续流转
