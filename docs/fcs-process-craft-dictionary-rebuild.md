# 工序工艺字典重构

## 范围

本次只收口工序工艺字典，不改技术包 BOM 页面、纸样管理页面、裁片明细、菲票打印和特殊工艺任务生成主流程。

## 准备阶段

准备阶段新增两个面料级工序：

- 缩水：`PREP_SHRINKING`，由物料清单中的缩水需求触发。
- 洗水：`PREP_WASHING`，由物料清单中的洗水需求触发。

洗水不再作为裁片部位级特殊工艺，不进入新的特殊工艺选择清单，也不作为裁片逐片特殊工艺生成任务。

## 特殊工艺分类

特殊工艺分为两类：

- 辅助工艺
- 特种工艺

辅助工艺清单：

- 绣花：裁片部位
- 打条：裁片部位
- 压褶：裁片部位
- 打揽：裁片部位
- 烫画：裁片部位
- 直喷：裁片部位
- 贝壳绣：裁片部位
- 曲牙绣：裁片部位
- 一字贝绣花：裁片部位
- 捆条：面料

特种工艺清单：

- 模板工序：裁片部位
- 激光开袋：裁片部位
- 特种车缝（花样机）：裁片部位
- 橡筋定长切割：辅料

捆条针对面料，后续由纸样层“添加捆条”维护，不在裁片逐片特殊工艺中选择。橡筋定长切割针对辅料，不在裁片逐片特殊工艺中选择。

## 裁床工序

裁床工序补齐三种裁法：

- 普通裁
- 激光定位裁
- 定向裁

流程口径：

```text
待裁剪
-> 普通裁 / 激光定位裁 / 定向裁
-> 裁剪完成
-> 菲票生成
-> 入裁片仓
```

本次只补齐字典，不改变裁床执行、菲票生成和裁片明细页面。

## 后续消费方式

后续技术包 BOM 可以通过准备阶段字典读取缩水、洗水需求字段；纸样管理可以读取裁片部位可选工艺和面料级捆条；菲票和特殊工艺任务生成可以读取目标对象与可生成任务标记，避免洗水被重复当作裁片部位特殊工艺。

## 流程图

```text
维护工序工艺字典
-> 准备阶段新增缩水和洗水
-> 特殊工艺按辅助工艺和特种工艺分类
-> 标记适用对象
-> 裁床工序补齐三种裁法
-> 技术包BOM和纸样管理后续读取字典
-> 菲票和特殊工艺任务后续消费字典
```

## 状态机

```text
待维护
-> 已补齐准备阶段
-> 已完成特殊工艺分类
-> 已标记适用对象
-> 已补齐裁床工艺
-> 字典可被技术包消费
```
