# FCS 工艺详情视图与后道工厂口径

## 流程图一：印花详情视图收口

```mermaid
flowchart LR
  A[生产单] --> B[印花加工单]
  B --> C[花型与调色]
  C --> D[打印转印]
  D --> E[送货交出]
  E --> F[审核记录]
  F --> G[执行进度]
  G --> H[异常与结算影响]
```

## 流程图二：染色详情视图收口

```mermaid
flowchart LR
  A[生产单] --> B[染色加工单]
  B --> C[打样备料]
  C --> D[染缸执行]
  D --> E[染色配方]
  E --> F[送货交出]
  F --> G[审核记录]
  G --> H[染色统计]
  H --> I[异常与结算影响]
```

## 流程图三：专门后道工厂

```mermaid
flowchart LR
  A[生产单] --> B[后道单]
  B --> C[后道]
  C --> D[质检]
  D --> E[复检]
  E --> F[后道交出仓]
  F --> G[交出记录]
  G --> H[接收方回写]
  H --> I[完成]
```

## 流程图四：非专门工厂含后道

```mermaid
flowchart LR
  A[生产单] --> B[车缝厂或其他工厂]
  B --> C[后道]
  C --> D[交给后道工厂]
  D --> E[后道工厂质检]
  E --> F[后道工厂复检]
  F --> G[后道交出仓]
  G --> H[交出记录]
  H --> I[接收方回写]
  I --> J[完成]
```

## 状态机一：印花加工单

```mermaid
flowchart LR
  A[待接单] --> B[待花型确认]
  B --> C[待调色]
  C --> D[等打印]
  D --> E[打印中]
  E --> F[打印完成]
  F --> G[待转印]
  G --> H[转印中]
  H --> I[转印完成]
  I --> J[待送货]
  J --> K[待回写]
  K --> L[待审核]
  L --> M[已完成]
  L --> N[已驳回]
  N --> J
```

## 状态机二：染色加工单

```mermaid
flowchart LR
  A[待接单] --> B[待样衣]
  B --> C[打样中]
  C --> D[打样完成]
  D --> E[待原料]
  E --> F[备料完成]
  F --> G[待排缸]
  G --> H[已排缸]
  H --> I[染色中]
  I --> J[脱水中]
  J --> K[烘干中]

  L --> M[打卷中]

  N --> O[待送货]
  O --> P[待回写]
  P --> Q[待审核]

  A --> E
  Q --> S[已驳回]
  S --> O
```

## 状态机三：专门后道工厂

## 状态机四：非专门工厂含后道

flowchart LR
