# 毛织纸样内部货号与毛织加工单搜索设计

## 背景

毛织纸样单据上存在类似 `2585` 的内部货号。该编号是 HiGood 内部按款式识别衣服的货号，同一款衣服唯一。毛织工厂现场和管理端需要用这个编号快速查找毛织加工单。

当前技术包的纸样包弹窗已经区分布料纸样和毛织纸样，毛织加工单页面也已有生产单、毛织单号、款式、任务类型、工厂和状态筛选。本次只补齐内部货号的录入、快照传递、搜索和列表展示，不做真实后端、权限、主数据中心或跨系统唯一性校验。

## 目标

1. 在技术包添加或编辑毛织纸样包时，允许填写内部货号。
2. 内部货号按 SPU 统一，同一 SPU 下不同技术包版本、多个毛织纸样包共享一个值。
3. 同一 SPU 多次填写内部货号时，以最后一次保存的非空值为准。
4. 毛织加工单可通过内部货号搜索，并在款式单元格中显示该值。
5. 毛织加工单统计卡片缩小为紧凑统计条，放在搜索条件下方，并随筛选结果联动。

## 字段口径

字段名称：`内部货号`

业务含义：

- 内部货号是 SPU / 款式级识别字段。
- 毛织纸样包只是录入入口，不代表每个纸样包拥有独立内部货号。
- 只有 `纸样类型 = 毛织纸样` 时展示该字段。
- 字段非必填。
- 保存毛织纸样包时，如果内部货号有非空值，则更新当前 SPU 的内部货号展示值。
- 同一 SPU 的多个技术包版本、多个毛织纸样包里出现不同内部货号时，原型按最后一次保存的非空值展示。

不做：

- 不阻断空值保存。
- 不做跨 SPU 重复校验。
- 不做历史版本差异查看。
- 不做独立内部货号维护页。

## 技术包页面设计

范围：添加或编辑纸样包弹窗。

展示规则：

- 当纸样类型为 `毛织纸样` 时，在基础信息区域展示 `内部货号` 输入框。
- 当纸样类型为 `布料纸样` 时隐藏该输入框。
- 输入框位于 `是否毛织` / `纸样分类` 附近，避免放到 Zip 文件或毛织部位明细区域。
- 占位文案使用 `例如：2585`。

保存规则：

- 保存纸样包时，把内部货号写入当前毛织纸样包数据。
- 如果内部货号非空，同步写入当前技术包内容的 SPU 级内部货号快照字段。
- 编辑旧毛织纸样包时，表单回显该纸样包保存过的内部货号；若纸样包自身为空，可回显当前 SPU 级内部货号，方便继续沿用。

## 毛织加工单设计

范围：`/fcs/craft/wool/work-orders`。

搜索：

- 保持现有筛选结构，不新增复杂筛选面板。
- 将款式搜索口径扩展为匹配 `款号 / 款名 / 内部货号`。
- 搜索输入的标签可调整为 `款式 / 内部货号`。

列表展示：

- 在“款式”单元格第二行展示内部货号。
- 建议展示格式：`内部货号：2585 / PO-xxx`。
- 若内部货号为空，不展示 `内部货号：-`，只保留生产单号，减少噪音。

统计：

- 页面顺序调整为：标题、搜索条件、紧凑统计条、加工单表格。
- 统计条样式参考截图中的小型灰底标签，不使用现有大 KPI 卡片。
- 统计按当前筛选结果计算，而不是按全量加工单计算。
- 最小统计项：
  - 加工单
  - 未接单
  - 待领料
  - 领料中
  - 待排机
  - 横机中
  - 待打印菲票
  - 计划数量
  - 完成数量

## 数据流

```mermaid
flowchart LR
  A["技术包 - 毛织纸样包填写内部货号"] --> B["保存纸样包"]
  B --> C["纸样包保留 internalStyleCode"]
  B --> D["当前 SPU 内部货号快照更新为最后一次非空值"]
  D --> E["生产单技术包快照"]
  E --> F["毛织任务 / 毛织加工单"]
  F --> G["毛织加工单搜索与款式单元格展示"]
```

## 状态与异常

- 内部货号为空：允许保存；毛织加工单不展示内部货号文本。
- 同一 SPU 多次填写：最后一次非空值覆盖当前展示值。
- 同一 SPU 多个纸样包存在旧值：不提示冲突，毛织加工单统一展示当前 SPU 值。
- 输入包含空格：保存时去除首尾空格。

## 验证要求

实现后需要验证：

1. 毛织纸样包弹窗展示内部货号输入框，布料纸样不展示。
2. 内部货号非必填，空值可保存。
3. 同一 SPU 下后保存的非空内部货号会覆盖毛织加工单展示值。
4. 毛织加工单可通过 `2585` 搜索到对应结果。
5. 款式单元格第二行展示 `内部货号：2585 / 生产单号`。
6. 毛织加工单统计条位于筛选条件下方，并按筛选结果变化。
7. 运行 `npm run check:prototype-design-governance`。
8. 运行最小页面或脚本验证，确认路由可达、无明显空白页和中文文案错误。
