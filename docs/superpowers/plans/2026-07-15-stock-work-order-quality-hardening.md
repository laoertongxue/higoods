# 库存建单与联合水溶状态机质量加固计划

## 目标

在不恢复独立工艺需求单的前提下，让染色、印花加工单从仓库真实库存创建，完整保留单位与计划完成时间，并恢复联合水溶染色的关键状态机回归覆盖。

## 实施步骤

1. 先补库存建单领域失败测试，覆盖库存不存在、名称或单位不匹配、数量无效、计划完成时间为空，以及成功后的单位和计划时间保真。
2. 复用现有仓库库存 Mock，收紧染色、印花库存建单函数与页面适配器，不再接受自由填写库存身份。
3. 补创建抽屉失败测试，显示就地错误、纠正或关闭后清除，并确保输入不会触发整页重渲染或重建抽屉。
4. 从删除前版本迁移联合水溶染色状态机检查，改用正式生产单染色加工单注册入口，不恢复需求单或伪需求数据。
5. 更新同一份原型审查记录，运行专项检查、构建、治理检查和 CodeGraph 同步，确认回归后提交。
