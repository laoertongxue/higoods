# FCS 交出回写与差异记录统一

本文档记录印花、染色、特殊工艺、后道工厂共用的交出回写、数量差异、审核记录和继续流转口径。当前原型没有后端服务，页面与移动端演示都读取前端统一事实源。

## 流程图

### 流程图一：统一交出回写流程

待交出仓记录
-> 工厂发起交出
-> 生成交出记录
-> 接收方回写实收数量
-> 系统计算差异数量
-> 数量一致
-> 更新交出记录为已回写
-> 进入下一环节或完成

### 流程图二：统一交出差异流程

交出记录
-> 接收方回写实收数量
-> 系统计算差异数量
-> 数量不一致
-> 生成差异记录
-> 生成审核记录
-> 平台处理差异
-> 确认差异继续流转或要求重新交出或关闭记录

### 流程图三：印花交出回写流程

印花加工单
-> 印花待交出仓
-> 印花工厂交出面料
-> 接收方回写实收面料米数
-> 数量一致进入审核或完成
-> 数量不一致生成印花差异记录
-> 平台处理差异

### 流程图四：染色交出回写流程

染色加工单
-> 染色待交出仓
-> 染厂交出面料
-> 接收方回写实收面料米数
-> 数量一致进入审核或完成
-> 数量不一致生成染色差异记录
-> 平台处理差异

### 流程图五：特殊工艺交出回写流程

特殊工艺单
-> 特殊工艺待交出仓
-> 工艺工厂交出裁片
-> 接收方回写实收裁片数量
-> 数量一致更新菲票状态
-> 数量不一致生成差异记录
-> 更新菲票数量变化和流转记录
-> 平台处理差异

### 流程图六：后道交出回写流程

后道单
-> 复检完成
-> 后道交出仓
-> 后道工厂交出成衣
-> 接收方回写实收成衣件数
-> 数量一致完成后道交出
-> 数量不一致生成后道差异记录
-> 平台处理差异

### 流程图七：非专门工厂后道限制流程

非专门工厂后道完成
-> 交给后道工厂
-> 生成后道工厂待加工仓记录
-> 后道工厂质检
-> 后道工厂复检
-> 后道交出仓
-> 后道工厂交出成衣

## 状态机

### 状态机一：统一交出记录

待回写
-> 已回写
-> 已关闭

差异流转：

待回写
-> 有差异
-> 平台处理中
-> 已回写
-> 已关闭

重新交出流转：

有差异
-> 需重新交出
-> 待回写

### 状态机二：统一差异记录

待处理
-> 处理中
-> 已确认差异
-> 已关闭

重新交出流转：

待处理
-> 需重新交出
-> 已关闭

### 状态机三：统一审核记录

待审核
-> 审核通过
-> 已关闭

异常流转：

待审核
-> 数量差异
-> 审核通过或审核驳回或已关闭

### 状态机四：特殊工艺差异处理

差异待处理
-> 更新菲票数量
-> 写入数量变化
-> 写入流转记录
-> 差异已确认

重新交出流转：

差异待处理
-> 要求重新交出
-> 待交出

### 状态机五：后道交出差异处理

后道待交出
-> 已交出
-> 待回写
-> 有差异
-> 平台处理中
-> 已确认差异或需重新交出
-> 已完成

## 统一对象与页面消费

- 统一交出记录：`ProcessHandoverRecord`
- 统一差异记录：`ProcessHandoverDifferenceRecord`
- 统一审核记录：`ProcessWarehouseReviewRecord`
- 工厂发起交出：`createProcessHandoverRecord`
- 接收方回写：`writeBackProcessHandoverRecord`
- 平台处理差异：`handleProcessHandoverDifference`
- 特殊工艺菲票差异同步：`applySpecialCraftDifferenceToFeiTickets`

印花、染色详情的送货交出、审核记录、异常与结算视图读取统一交出、审核、差异记录。特殊工艺详情的绑定菲票、数量变化、差异上报、流转记录通过统一差异记录追溯到菲票变化。后道交出仓只展示复检完成后生成的待交出仓记录，并通过统一交出记录承接回写与差异。

差异处理只记录处理结果和后续动作，不直接生成质量扣款流水、对账流水或结算流水。
