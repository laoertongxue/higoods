二、运行时数据流

sequenceDiagram participant User as 👤 用户 participant DOM as 🌐 #app participant Main as ⚡ main.ts participant Handler as 🎯 Handler (FCS/PCS/PDA) participant Data as 🗄️ data/ (Mock) participant Store as 📦 AppStore participant Router as 🔀 resolvePage() participant Page as 📄 renderXxxPage() User->>DOM: 点击 / 输入 / 提交 DOM->>Main: 事件冒泡 (click/input/submit) Main->>Handler: 按路径前缀分派 alt 需要修改数据 Handler->>Data: 读写 Mock 数据 Data-->>Handler: 更新结果 end Handler->>Store: 更新状态 (pathname/tabs) Store->>Router: 通知路径变更 Router->>Page: 调用对应 render 函数 Page-->>Router: 返回 HTML 字符串 Router-->>DOM: root.innerHTML 挂载 DOM-->>User: 页面重新渲染
