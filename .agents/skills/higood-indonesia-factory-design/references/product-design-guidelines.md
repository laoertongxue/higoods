## Non-Negotiables

- Role first: 管理端 / 主管端 / 员工执行端.
- Frontline pages must not expose internal system concepts.
- One frontline page should have one primary action.
- System calculates quantities and differences.
- Errors must say how to fix the problem.
- UI style must support现场识别 and防错, not decoration.
- Mock data must show upstream, downstream, exceptions, differences, and traceability.
